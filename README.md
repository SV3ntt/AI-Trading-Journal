# AI Trading Journal

An AI-powered trading journal built in Python.

## Overview

The goal of this project is to help traders:

- Record and organize trades
- Track win rate and performance
- Analyze risk-to-reward ratios
- Identify common trading mistakes
- Receive AI-generated feedback and insights

This project is being built as a learning project and potential startup idea.

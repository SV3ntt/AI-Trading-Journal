📈 AI Trading Journal
A command-line trading journal built in Python to help traders record, analyze, and improve their trading performance.

This project began as a way to learn Python programming while creating a practical tool for my own trading. It has since evolved into a feature-rich trading journal and serves as the foundation for a future AI-powered trading analytics platform.
